# Life Management System
Life Management System je Java Swing desktop aplikacija koja omogućava korisnicima praćenje ličnih navika i svakodnevnih aktivnosti kroz više jednostavnih trackera. Aplikacija je namijenjena ličnoj organizaciji i unapređenju zdravih životnih navika.
Aplikacija podržava više korisnika, autentikaciju (prijavu i registraciju), upravljanje korisničkim računom i praćenje osnovnih životnih aktivnosti.

## Korištene tehnologije
- Java (Swing GUI)
- IntelliJ IDEA
- IntelliJ IDEA GUI Designer
- Git & GitHub

## Pokretanje aplikacije
- Otvoriti projekat u IntelliJ IDEA
- Provjeriti da je podešena Java SDK verzija 17
- Pokrenuti klasu Main
- Aplikacija se pokreće sa početnim ekranom za Login ili Registraciju

## Autentikacija korisnika
- Aplikacija omogućava:
- Registraciju novog korisnika
- Prijavu postojećeg korisnika
- Svaki korisnik ima vlastite podatke

## Trackeri
- Aplikacija sadrži sljedeće trackere:
  Spavanje
- Unos broja sati spavanja
  Trening
- Evidencija trening aktivnosti
  Ishrana
- Unos osnovnih informacija o ishrani
  Voda
- Praćenje dnevnog unosa vode
-Evidencija količine popijene vode
- Podsticanje redovnog unosa tečnosti
Finance App 
- Finance App je dodatni modul aplikacije Life Management System koji omogućava korisnicima praćenje ličnih finansija na jednostavan i pregledan način.

## Grafički interfejs
- Aplikacija koristi Java Swing komponente
- Poruke i obavještenja korisniku se prikazuju putem JLabel
- Jednostavan i pregledan dizajn
- Fokus na lakoću korištenja

## Arhitektura aplikacije
- Aplikacija je organizovana kroz jasno odvojene dijelove:
- GUI forme (Swing)
- Logika aplikacije
- Upravljanje korisničkim podacima

## Sigurnost i privatnost
- Podaci su vezani za konkretnog korisnika
- Svaki korisnik vidi samo svoje unose
- Brisanje korisničkog računa uklanja korisničke podatke
